# SunoAI Extension

SunoAIサイトから楽曲情報を取得・保存するChrome拡張機能です。

## 機能

- **楽曲情報の保存**: SunoAIのページからタイトル、歌詞、Styleプロンプト、タグなどの情報を取得してローカルに保存
- **URL収集**: ワークスペース内の全URLを取得して保存
- **右クリックメニュー**: SunoAIのページで右クリックして簡単にデータを保存
- **データエクスポート**: 保存したデータをJSON形式でエクスポート
- **データ管理**: ポップアップから保存済みデータの閲覧・削除が可能

## インストール方法

1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. このプロジェクトのフォルダを選択

## 使い方

### 楽曲情報を保存する

1. SunoAIのページを開く
2. 右クリック → 「楽曲情報を保存」を選択
   または
   拡張機能のアイコンをクリック → 「現在のページを保存」ボタンをクリック

### URLを取得する

1. SunoAIのワークスペースページを開く
2. 右クリック → 「ワークスペース内の全URLを取得」を選択
   または
   拡張機能のアイコンをクリック → 「URL一覧」タブ → 「URLを取得」ボタンをクリック

### データをエクスポートする

1. 拡張機能のアイコンをクリック
2. 「アクション」タブ → 「全データをエクスポート」をクリック
3. JSONファイルがダウンロードされます

## ファイル構成

```
SunoExtension/
├── manifest.json       # 拡張機能の設定ファイル
├── content.js         # ページに注入されるスクリプト（データ取得）
├── background.js      # バックグラウンドスクリプト（データ保存・管理）
├── popup.html         # ポップアップのHTML
├── popup.css          # ポップアップのスタイル
├── popup.js           # ポップアップのロジック
├── icons/             # アイコンファイル（要作成）
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── README.md          # このファイル
```

## 注意事項

- この拡張機能は**補助的な機能のみ**を提供します
- 自動でMP3ファイルをダウンロードする機能は**含まれていません**（自動化botの扱いになるため）
- Chromeに流れてきたデータを補足して保存する機能に留めています

## 開発

### 必要な権限

- `storage`: データのローカル保存
- `contextMenus`: 右クリックメニューの追加
- `activeTab`: 現在のタブへのアクセス
- `notifications`: 通知の表示
- `downloads`: データのエクスポート

### カスタマイズ

`content.js`の`extractSongData()`関数を編集することで、SunoAIのサイト構造に合わせてデータ取得ロジックを調整できます。
