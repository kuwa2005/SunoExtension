# アイコンファイル

このフォルダには以下のアイコンファイルが必要です：

- `icon16.png` (16x16ピクセル)
- `icon48.png` (48x48ピクセル)
- `icon128.png` (128x128ピクセル)

## アイコンの作成方法

1. 任意の画像編集ソフト（Photoshop、GIMP、Canvaなど）でアイコンを作成
2. 上記のサイズでPNG形式で保存
3. このフォルダに配置

## 一時的な解決策

アイコンがない場合、Chrome拡張機能は警告を表示しますが、動作には影響しません。
後からアイコンを追加することができます。
